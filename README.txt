Проект: Веб‑страница для группы ATK 24-51
Включено:
- index.html — основной файл (улучшённая версия)
- styles.css — стили
- script.js — логика: localStorage, модалки, добавление/редактирование новостей, загрузка фото в галерею
- assets/* — плейсхолдеры изображений (.svg)

Что было улучшено:
- Отделены CSS и JS, улучшена структура;
- Доступность: skip-link, aria-атрибуты, keyboard-friendly;
- Сохранение новостей и галереи в localStorage (не уходят при перезагрузке);
- Редактирование и удаление новостей, добавление картинок через URL или загрузку файла;
- Загрузка нескольких фото в галерею + просмотр в fullscreen;
- Уведомления (toasts) и улучшённая отзывчивость.

Как запустить:
1. Распакуйте zip.
2. Откройте index.html в браузере (не требуется сервер для базовых функций).
3. Для работы с локальными файлами: браузер может блокировать FileReader для локальных файлов в некоторых настройках — в таком случае откройте через локальный сервер (например: python -m http.server).

Дополнительно: если хотите, могу:
- подключить деплой на GitHub Pages,
- добавить бэкенд (Node/Express) для хранения новостей и фото,
- настроить CI (GitHub Actions) и инструкции по работе группой через Git.
